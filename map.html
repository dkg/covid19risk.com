GPS Heat Map Demo

We plan to build a web demo showing the kind of information we could display if we had the technology to build a heat map based on sparse GPS data. An app with this heat map would give people a better idea of current risk levels in their local area. We hope to supplement our data with openly available Johns Hopkins data, which updates every day, with the app providing some additional information at higher resolution.

Our Plan for a Demo

We plan to build a web demo using a small GPS trajectory dataset, which was collected by the Microsoft Research Asia Geolife project by 182 users in a period of over five years from April 2007 to August 2012. We hope to use this small dataset to help validate demo simulations, and if we can demonstrate the method on a small scale, we could apply it to a larger dataset like the BBC Four Pandemic dataset.
