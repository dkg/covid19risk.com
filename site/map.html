GPS Heat Map Demo

We've built a web demo available here which demonstrates the basic capacity to create a risk heatmap that changes over time. This map doesn't show actual COVID-19 spread: it was simulated using a small GPS trajectory dataset, which was collected by the Microsoft Research Asia Geolife project by 182 users in a period of over five years from April 2007 to August 2012. We hope to use the GeoLife trajectories to help validate demo simulations, and if we can demonstrate the method on a small scale, we could apply it to a larger dataset like the BBC Four Pandemic dataset.

Eventually, our goal is to integrate both publicly available data on COVID-19 cases and user-shared trajectories to create similar heatmaps globally. We want to be confident that these heatmaps are accurate and helpful before we release them, though: a badly-implemented map may do more harm than good.

The code behind our proof-of-concept modelling so far is available on GitHub here.
